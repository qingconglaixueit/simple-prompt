# AI提示词助手 Chrome插件

一个可配置多个提示词对接SiliconFlow AI接口的浏览器插件，支持主题切换，提供未来机器人和优雅少女两种风格。

## 功能特性

### 核心功能
- ✅ **可配置提示词**：支持添加、删除、编辑多个提示词
- ✅ **AI解析**：将用户输入与提示词组合后发送到SiliconFlow API
- ✅ **实时时间**：自动添加当前时间到提示词最前面
- ✅ **结果复制**：一键复制AI解析结果
- ✅ **主题切换**：支持两种风格主题

### 主题风格
1. **未来机器人风格**：深色背景、霓虹色彩、科技感字体、网格背景
2. **优雅少女风格**：浅色背景、粉色紫色渐变、优雅衬线字体、柔和阴影

## 安装方法

### 本地安装
1. 克隆仓库到本地：
   ```bash
   git clone git@github.com:qingconglaixueit/simple-prompt.git
   ```

2. 打开Chrome浏览器，进入扩展程序管理页面：
   - 在地址栏输入：`chrome://extensions/`
   - 开启右上角的"开发者模式"

3. 加载已解压的扩展程序：
   - 点击"加载已解压的扩展程序"
   - 选择克隆的仓库目录

4. 安装完成后，插件图标将显示在Chrome工具栏

## 使用说明

### 配置步骤
1. 点击插件图标，选择"配置提示词"
2. 在配置页面设置：
   - SiliconFlow模型名称（例如：gpt-4o-mini）
   - SiliconFlow API Key
   - 添加一个或多个提示词

### 使用步骤
1. 点击插件图标，选择要使用的提示词
2. 在输入框中输入待处理的内容
3. 点击"AI解析"按钮
4. 等待AI处理完成后，查看结果
5. 点击"复制结果"按钮复制解析结果

## 提示词配置规则

### 提示词组合方式
- 如果提示词中包含`{content}`占位符，则会将用户内容替换到该位置
- 如果没有占位符，则将用户内容追加到提示词尾部

### 示例提示词
```
请将以下内容润色得更专业：{content}
```

或
```
请总结以下内容的主要观点
```
